## 🛡️ Mini-Project: Quantum-Randomness-Lab

As part of the Day 1 to 10 milestone, I built a **Quantum Randomness Laboratory**. This project explores how true randomness is generated by collapsing quantum wave functions. 

As a **Unique Selling Proposition (USP)**, I also engineered a **Quantum-Secured Password Generator** that maps quantum superposition states to character arrays to create unbreakable, truly random passwords.

### 🚀 How to Run the Mini-Project

1. Navigate to the Mini Project directory:
   ```bash
   cd "Mini_Project_1/Quantum-Randomness-Lab"
   ```
2. Install the required quantum libraries:
   ```bash
   pip install qiskit qiskit-aer matplotlib numpy scipy
   ```

### 🔬 Available Scripts and Features

Run these commands inside the `Quantum-Randomness-Lab` folder to test the different features:

#### 1. Basic Quantum Random Numbers
Generates raw random numbers (`0` and `1`) using a 1-qubit quantum circuit.
```bash
python qrng.py
```

#### 2. Visual Histogram (Plotting Randomness)
Displays a matplotlib bar chart (Histogram) showing the frequency distribution of the quantum outcomes.
```bash
python histogram.py
```
> 🔗 [View Histogram Output Image](assets/Histogram.png)

#### 3. Statistical Analysis
Applies mathematical analysis to calculate the **Mean** and **Standard Deviation** of the generated quantum sequence.
```bash
python statistics.py
```

#### 4. Classical vs Quantum Comparison
Generates a classical pseudo-random number sequence to contrast with quantum probability.
```bash
python comparison.py
```

#### 5. 🌟 USP: Quantum-Secured Password Generator
Generates a 16-character ultra-secure password using a 6-qubit quantum circuit. 
```bash
python quantum_password.py
```

For the second milestone, I built a production-grade full-stack **Quantum Computing Management Dashboard**. This platform allows researchers to design circuits, execute them on simulated or IBM backends, and monitor real-time quantum hardware telemetry.

### 🌟 Key Features
* **Real-time Quantum Telemetry**: Live tracking of QPU temperatures and queue loads using animated sparklines.
* **High-Fidelity Circuit Rendering**: Native integration with IBM Qiskit `matplotlib` drawer for professional, color-coded quantum circuit export (PNG).
* **State Machine Execution Engine**: Reliable quantum job tracking (Draft ➔ Queued ➔ Running ➔ Completed) managed via FastAPI.
* **Enterprise Dashboard**: Live data-driven KPIs, execution history, and analytical telemetry.

### 🚀 How to Run Mini-Project 2

**1. Start the Backend (FastAPI)**
```bash
cd "Mini_Project_2/quantum-platform-enterprise/backend"
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python -m uvicorn main:app --reload --port 8000
```

**2. Start the Frontend (Next.js)**
```bash
cd "Mini_Project_2/quantum-platform-enterprise/frontend"
npm install
npm run dev
```
Visit `http://localhost:3000` to access the Enterprise Dashboard.
